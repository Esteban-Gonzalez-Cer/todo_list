# JS ToDo List - Peer Programming Project

## Descripción
Aplicación web dinámica para la gestión de tareas (ToDo List) desarrollada con Vanilla JavaScript. Este proyecto consiste en la implementación de la lógica interactiva sobre un template estático en HTML y Bootstrap, creando un CRUD completo, persistencia de datos y filtros de búsqueda mediante la metodología de **Peer Programming**.

## Características Principales
* **CRUD de Tareas:** Creación, lectura, actualización (edición de texto y cambio de estado) y eliminación de tareas.
* **Persistencia de Datos:** Integración con `localStorage` para mantener las tareas guardadas tras recargar el navegador.
* **Filtros de Búsqueda:** Búsqueda dinámica por palabras clave y filtrado rápido por estado (Todas, Completadas, Pendientes).
* **Innovación Propuesta:** Asignación de niveles de prioridad (Alta, Media, Baja) con indicadores visuales en la interfaz.

## Tecnologías Utilizadas
* **Estructura y Estilos:** HTML5, CSS3, Bootstrap 4.5.2 (Tema Darkly), FontAwesome.
* **Lógica:** Vanilla JavaScript (ES6+), manipulación del DOM.
* **Almacenamiento:** Web Storage API (`localStorage`).
* **Control de Versiones y Colaboración:** Git y GitHub.
